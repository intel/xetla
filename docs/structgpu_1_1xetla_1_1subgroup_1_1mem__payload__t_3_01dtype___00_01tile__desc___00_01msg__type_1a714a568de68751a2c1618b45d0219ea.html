Detailed Description

template<typename dtype_, typename tile_desc_, mem_layout mem_layout_, gpu_arch arch_tag_>
struct gpu::xetla::subgroup::mem_payload_t< dtype_, tile_desc_, msg_type::block_2d, mem_layout_, mem_space::global, arch_tag_, std::enable_if_t<(arch_tag_==gpu_arch::Xe)> >

Is to describe the global memory surface for block-2d load/store for each block in one tile, a payload message is prepared here.

in tile_load case, memory transpose, register transpose, memory transform and dword transpose can be enable. While in tile store case, we only support row major store, no register operations can be applied.
