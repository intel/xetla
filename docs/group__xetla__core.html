Detailed Description

This is a low-level API wrapper for ESIMD.

Some terminologies used in the API documentation:

  • word - 2 bytes.
  • dword ("double word") - 4 bytes.
  • qword ("quad word") - 8 bytes.
  • oword ("octal word") - 16 bytes.
