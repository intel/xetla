Detailed Description

template<typename dtype_, typename tile_desc_>
struct gpu::xetla::subgroup::mem_payload_t< dtype_, tile_desc_, msg_type::unaligned_2d, mem_layout::row_major, mem_space::global, gpu_arch::Xe >

Is to describe the global memory surface for unaligned-2d load/store for each block in one tile, a payload message is prepared here.

in tile_load case, memory transpose, register transpose, memory transform and dword transpose can be enable. While in tile store case, we only support row major store, no register operations can be applied.
