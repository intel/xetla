Detailed Description

template<typename dtype_in_, typename dtype_out_, typename dtype_acc_, typename reduction_attr_, typename fused_op_t_>
struct gpu::xetla::kernel::xetla_row_reduction_t< dtype_in_, dtype_out_, dtype_acc_, reduction_attr_, gpu_arch::Xe, fused_op_t_ >

Is the row_reduction functor for Xe The idea is threads in group will cooperatively process matrix_m x wg_tile_n.

Each time, each thread will load sg_tile_m x sg_tile_n data into register and do the reduction. A dynamic counter will be set in SLM to solve load imbalance problem.

Template Parameters
dtype_in_Is the data type of input.
dtype_out_Is the data type of output.
dtype_acc_Is the accumulation data type.
reduction_attr_Is the tile size for each group to do the reduction.
fused_op_t_Is the fused op functor.

Main execution function for row reduction.

The basic process is 1) load data -> reduction 2) data sharing via SLM -> reduction 3) write out.
