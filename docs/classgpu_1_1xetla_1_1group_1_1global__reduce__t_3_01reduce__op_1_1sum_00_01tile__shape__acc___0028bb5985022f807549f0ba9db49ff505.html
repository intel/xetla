Global reduction.

1) each group stores tile data to global memory by using global atomic add -> 2) after reduction complete, update the counter by using atomic inc -> 3) the last group load the data back to EU, and clean the accumulation buffer and counter buffer.

Note
only the last group has the valid data.
Parameters
gIs the workgroup of the current tile.
matAccIs the input of the reduction.
mem_desc_accIs the memory descriptor of accumulation buffer.
mem_desc_cntIs the memory descriptor of counter buffer.
slm_baseIs the slm base address.
nbarrier_baseIs the named barrier base.
