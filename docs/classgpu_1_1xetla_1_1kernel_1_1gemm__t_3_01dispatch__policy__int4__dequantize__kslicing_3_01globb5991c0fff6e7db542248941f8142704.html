Main execution function for GEMM.

The processing order is 1) set group-level base and boundary, split group to workgroups -> 2) local_kslicing_ratio x brgemms -> 3) local kslicing -> 4) local_kslicing_ratio x epilogues.

Parameters
eiIs the execution item, returns execution related information, such as workgroup id, subgroup id...
argsIs the GEMM arguments for application-related runtime variables.
slm_baseIs the slm base address.
nbarrier_baseIs the named barrier base.
