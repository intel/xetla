Detailed Description

template<typename dtype_in_, typename dtype_out_, typename dtype_compute_, typename data_transformer_attr_, mem_layout mem_layout_in_, int need_fp8_op>
struct gpu::xetla::kernel::xetla_data_transformer< dtype_in_, dtype_out_, dtype_compute_, data_transformer_attr_, mem_layout_in_, need_fp8_op, gpu_arch::Xe >

Is the data_transformer functor for Xe Each time, each thread will load sg_tile_m x sg_tile_n data into register and do the data convert.

Template Parameters
dtype_in_Is the data type of input.
dtype_out_Is the data type of output.
dtype_compute_Is the data type of calculation.
data_transformer_attr_Is the tile size for each group to do the data convert.
mem_layout_in_Indicates the input data col major or row major.
need_fp8_opIndicates whether fp8-related operations are required.
