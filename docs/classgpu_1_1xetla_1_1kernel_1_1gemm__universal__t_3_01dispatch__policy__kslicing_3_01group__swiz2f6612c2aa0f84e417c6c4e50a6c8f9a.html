Main execution function for GEMM_UNIVERSAL.

The processing order is 1) set group-level base and boundary, split group to workgroups -> 2) num_local_kslicing x gemms -> 3) local kslicing -> 4) num_local_kslicing x epilogues.

Parameters
itemIs the sycl::nd_item, returns execution related information, such as workgroup id, subgroup id...
argsIs the GEMM_UNIVERSAL arguments for application-related runtime variables.
slm_baseIs the slm base address.
nbarrier_baseIs the named barrier base.
