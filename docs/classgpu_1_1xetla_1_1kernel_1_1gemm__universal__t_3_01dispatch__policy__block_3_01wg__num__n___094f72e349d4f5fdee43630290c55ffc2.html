Main execution function for GEMM_UNIVERSAL.

The processing order is 1) set group-level base and boundary -> 2) gemm -> 3) epilogue.

Parameters
eiIs the execution item, returns execution related information, such as workgroup id, subgroup id...
argsIs the GEMM_UNIVERSAL arguments for application-related runtime variables.
slm_baseIs the slm base address.
nbarrier_baseIs the named barrier base.
