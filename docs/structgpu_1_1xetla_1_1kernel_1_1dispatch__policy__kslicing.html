Detailed Description

template<int global_ratio_ = 1, int local_ratio_ = 1, gpu_arch arch_tag_ = gpu_arch::Xe>
struct gpu::xetla::kernel::dispatch_policy_kslicing< global_ratio_, local_ratio_, arch_tag_ >

Kslicing GEMM implementation.

A special GEMM implementation to increase the hardware occupancy by splitting the GEMM task along k dimension. It includes inter-group reduction (by using global atomic) and intra-group reduction (by using local memory for data exchange).

Template Parameters
global_ratio_Is the k dim split ratio between groups.
local_ratio_Is the k dim split ratio within a group.
arch_tag_Is the HW architecture.
