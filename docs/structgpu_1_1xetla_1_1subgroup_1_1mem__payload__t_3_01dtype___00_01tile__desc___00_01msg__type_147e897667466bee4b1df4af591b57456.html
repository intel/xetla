Detailed Description

template<typename dtype_, typename tile_desc_>
struct gpu::xetla::subgroup::mem_payload_t< dtype_, tile_desc_, msg_type::block_1d, mem_layout::row_major, mem_space::global, gpu_arch::Xe >

Is to describe the global memory surface for block-1d load/store For a block-1d payload message we need to set the base address and offset of surface.

memory transpose and register operations can not be applied here. 2d and 1d surface both can be accepted here
