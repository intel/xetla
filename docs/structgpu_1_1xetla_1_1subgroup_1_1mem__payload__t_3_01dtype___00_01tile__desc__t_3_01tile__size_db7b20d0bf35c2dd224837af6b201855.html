Is to describe the shared local memory surface for scattering store.

  1. data located in shared local memory 2. tile will be stored in scatter mode
  2. data in register file is vnni packed and col majored.
    Template Parameters
    dtypeIs the data type
    tile_desc_Is the tile descriptor
    mem_layout_Is the memory layout
